# ❗ 프로젝트 후기
### ❣️ 좋았던점
 + 이번 프로젝트를 통해 요양원 홈페이지라는 실제 서비스 목적을 가진 웹 사이트를 처음부터 끝까지 구현해볼 수 있었습니다. 단순한 화면 제작이 아니라, 가족과 요양보호사라는 서로 다른 사용자 역할에 따라 기능과 권한을 분리하는 구조를 직접 설계해볼 수 있었던 점이 인상 깊었습니다.
 + 로그인 시 사용자 유형(가족 / 요양보호사 / 관리자)에 따라 접근 가능한 기능을 다르게 구성하고, 게시글 작성·수정·삭제 권한을 제한하는 과정에서 웹 서비스에서 권한 관리가 왜 중요한지 명확히 이해할 수 있었습니다.
 + 또한 회원가입 시 입력값 검증, 아이디 중복 확인, 비밀번호 정규식 적용 등을 직접 구현하며, 사용자 데이터의 무결성과 보안 기초 개념에 대해 실습을 통해 학습할 수 있었습니다.
 + 게시판 기능을 구현하면서 검색, 글 등록, 수정, 삭제 흐름을 직접 연결해보며, 프론트엔드와 서버, DB가 어떻게 하나의 흐름으로 동작하는지 전반적인 구조를 이해하게 되었다는 점에서 큰 성취감을 느꼈습니다.

### 💔 아쉬운점
 + 개발 기간과 학습 단계의 한계로 인해, 프로젝트 전반에서 게시판 기능 위주로 구현이 진행되었고, 게시글 수정 기능과 일부 메뉴들은 끝까지 완성하지 못한 점이 아쉬움으로 남았습니다.
특히 기능 구현 범위를 초기에 명확하게 정하지 못해, 전체 구조를 끝까지 다듬지 못한 점에서 아쉬움을 느꼈습니다.
 + 이번 프로젝트를 통해 기능의 개수보다 하나의 기능을 끝까지 완성도 있게 구현하는 것이 더 중요하다는 점을 체감하게 되었으며,
이후 프로젝트에서는 기획 단계에서 구현 범위를 보다 명확히 설정하고, 핵심 기능부터 단계적으로 완성해 나가는 방식으로 개발을 진행할 계획입니다.
또한 이번에 미완성으로 남은 기능들은 개인적으로 보완하며, 실제 서비스 흐름에 맞는 구조로 개선해 나가고 싶습니다.
